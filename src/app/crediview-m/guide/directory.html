    • pages: 크레디뷰 서비스 상세페이지 스타일 디렉토리
      • _board.scss: 게시판 style
      • _box.scss: wrapper 및 box style
      • _card.scss: 인증서 card, 등급 산출내역 card style
      • _common.scss: 기타 공통 style
      • _console.scss: 콘솔 서비스 화면 공통 style
      • _doc.scss: 신용평가등급확인서 style
      • _guide.scss: 신용평가체계 안내 및 개인정보처리방침 style
      • _members.scss: 로그인 및 회원가입페이지 style
      • _service.scss: 기업신용평가 메뉴 style
      • _table.scss: table 및 관련 style
      • _welcome.scss: 웰컴페이지 style
    • landing: 크레디뷰 서비스 랜딩페이지 스타일 디렉토리
      • _index.scss: 랜딩페이지 layout style
      • _section.scss: 랜딩페이지 섹션별 style
